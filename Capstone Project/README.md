## Theme
This is a website showing Modern stylist fashion e-commerce website.
## Features
# 1.Testimonials:
“What Our Customers Say” section with three comments,to help the customers for good review.
# 3.Products/Services Section:
High quality products and easy to add cart option.
# 4.Newsletter Signup:
Form to collect emails, centered with CTA button.
# 5.Footer:
Multi-column grid with navigational links, social media links, hover effects, and smooth transitions.
# 6.Global Transitions:
Smooth animations applied for hover, color, shadow, transform, and other interactive elements.
## Breakpoints
1.Mobile: 0px – 767px → Single-column layout, smaller paddings, stacked sections.
2.Tablet: 768px – 1023px → 2-column layout for cards, slightly reduced padding.
3.Desktop: 1024px+ → Full multi-column layout, larger spacing, horizontal card rows.
# How to run/view 
# 1.Open in Browser:
Save all your HTML, CSS, and JS files in the same folder.
Open the HTML file in any browser (Chrome, Firefox, Edge).
# 2.Open your project folder in VS Code → Right-click HTML file → “Open with Live Server.”
# 3.Responsive Testing:
Resize browser window or use DevTools (F12 → Toggle Device Toolbar) to check breakpoints.
