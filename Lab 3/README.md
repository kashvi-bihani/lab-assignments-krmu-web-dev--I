## Lab title 
Responsive Personal Portfolio Website
## Description 
This webpage is a more advanced version of personal portfolio containing animations, flex, grids, etc.

## Steps to view responsive behaviour 

1.Open your website in a browser (like Chrome, Firefox, or Edge).
2.Open developer tools:
In Chrome: Press F12 or Ctrl+Shift+I
Enable responsive design mode:
3.Click the device icon (looks like a phone/tablet) in developer tools.
4.Select different devices or set custom screen widths:
For example: iPhone, iPad, or custom widths like 480px, 768px, 1024px.
5.Observe changes in layout, images, menus, and content as screen width changes.
6.Check for issues: overlapping text, hidden buttons, broken layout, etc.

## Breakpoints 
Common ones:
Mobile: 320px – 480px
Tablet: 481px – 768px
Small desktop / laptop: 769px – 1024px
Large desktop: 1025px and above
