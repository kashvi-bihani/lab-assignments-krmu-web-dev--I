# JavaScript Quiz Game

This project is a simple quiz game built using JavaScript.  
The quiz runs in the browser console using `prompt()` and `alert()`.

## Features
- 5 quiz questions in multiple-choice format (A/B/C/D)   
- Immediate feedback (correct/wrong + hints)  
- Score tracking  
- High-score saved using `localStorage`  
- Restart option
